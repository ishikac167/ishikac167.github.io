HoPE - Health Predictor

Project information

  • Category: Web & App design
  • Project date: June 2022
  • Web Project URL: Repository Link
  • App Project URL: Repository Link

Problem Definition: Extract the information about preliminary symptoms and adapt an algorithm using machine learning techniques to predict certain medical ailments. If an ailment can be predicted, then early treatment can be given to the patients which can save the life of patients. We also intend to provide doctor and blog recommendation to the patients for swifter treatment.

Objective:

  • Make technology more useful in solving problematic healthcare challenges.
  • We aspire to enhance public health.
  • The current lifestyle of humans has seen a drastic change over the past few years. Keeping this in mind we have chosen the folliwng medical ailments:
    1. Myopia
    2. Colour Blindness
    3. Polycystic Ovarian Syndrome / Disease (PCOS / PCOD)
    4. Probability of Heart Attack
